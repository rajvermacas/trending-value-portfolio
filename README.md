# Conda

## Create environment
```bash
conda env create -f conda-env.yml
```

## Activate environment
```bash
conda activate env-name
```

## Deactivate environment
```bash
conda deactivate
```

## Remove environment
```bash
conda remove -n env-name --all
```

## Export environment
```bash
conda env export > conda-env.yml
```
